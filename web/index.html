推荐流程

  1. 在页面里填写店铺、站点和 ASIN，或导入原来的 .xlsx / .csv。
  2. 批量录入建议整理成“店铺、站点、ASIN”三列 CSV，然后用上方表格文件导入。
  3. 当前清单会自动保存在本机浏览器里，刷新页面后可继续；也可以导出 CSV 备份。
  4. 店铺 JPG 由业务员每次从“店铺图片”入口手动选择，可一次多选。
  5. Chrome / Edge 优先点“保存到文件夹”；不支持直接保存时再用“下载 ZIP（备用）”。生成结果会统一放进一个按日期时间命名的批次文件夹。

浏览器限制

浏览器不能无授权写入网页所在根目录，也不能预填任意本机路径或打开系统文件管理器。Chrome / Edge 可以直接保存到用户选择的普通子文件夹；Safari / Firefox 使用 ZIP 备用。
