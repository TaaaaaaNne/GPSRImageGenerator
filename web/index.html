推荐流程

  1. 在页面里填写店铺、站点和 ASIN，或导入原来的 .xlsx / .csv。
  2. 常用店铺 JPG 可以放进内置素材库，网页打开后会自动加载。
  3. 临时图片也可以一次选择整个素材文件夹。
  4. 第一次点击“保存到文件夹”时会从下载目录开始选择；请在下载里新建或选择一个普通子文件夹，例如 GPSR输出。

浏览器限制

浏览器不能无授权写入网页所在根目录，也不能预填任意本机路径或打开系统文件管理器。Chrome / Edge 通常不允许直接选择“下载”根目录本身；请选择下载目录里的普通子文件夹。如果权限被系统收回，页面会让你重新选择。
