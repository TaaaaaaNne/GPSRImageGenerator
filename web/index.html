推荐流程

  1. 在页面里填写店铺、站点和 ASIN，或导入原来的 .xlsx / .csv。
  2. 常用店铺 JPG 可以放进内置素材库，网页打开后会自动加载。
  3. 临时图片也可以一次选择整个素材文件夹。
  4. 桌面版默认保存到系统下载目录里的 GPSR输出 文件夹。
  5. 保存完成后可以直接打开输出文件夹。

桌面版说明

Windows / macOS 桌面版会显示完整输出路径，并能直接打开输出文件夹。如果用普通浏览器打开 index.html，仍会保留 ZIP 下载和浏览器文件夹保存备用。
